<h4>Manual Testing for eCommerce Website</h4>

<p align="justify"> This repo contains my manual testing work for an eCommerce web application.
I created detailed Test Cases, an RTM (Requirement Traceability Matrix), and covered essential functionalities such as user registration, login, home page, product search, and product details. </p>

<b>Repository Contents</b>  
1. eCommerce_manual testing (Excel) _  
A comprehensive set of manual test cases covering multiple modules or the application:  
-Register Module  
-Home Page  
-Search Functionality  
-Product Details Page  
-Cart Page  
-Checkout, Payments Page  
-My Orders Page  
-Customer Service Page  
  
It also includes columns such as: Test Case ID, Test Scenario, Preconditions, Test Steps, Expected Result, Actual Result, Status.   

2. Requirement Traceability Matrix (RTM)  
RTM includes:  
-Requirement IDs  
-Test Scenario IDs  
-Scenario Descriptions  
-Mapped Test Case IDs  

<p align="justify"> Objective of this Repo is to demonstrate my understanding and skills in test case design, requirement analysis, traceability coverage, UI/UX validation, 
  Positive/Negative Testing as well as my understanding the flow of an eCommerce application. </p>
